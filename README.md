# 🌧️ Rain Alarm Circuit

A simple and cost-effective **Rain Detection Alarm System** built using a **555 Timer IC**, **Rain Sensor**, and **BC547 Transistor**. The system detects rainfall and instantly alerts users through a **buzzer** and **LED indicator**.

## 🚀 Features

* Real-time rain detection
* Audio and visual alerts
* Low-cost and easy-to-build circuit
* Custom PCB design and fabrication
* Proteus simulation and 3D PCB visualization

## 🛠️ Components Used

* 555 Timer IC
* Rain Sensor
* BC547 NPN Transistor
* Buzzer
* Red LED
* 10µF Capacitor
* Resistors (100kΩ, 4.7kΩ, 1kΩ, 470Ω)
* 9V Battery

## ⚙️ Working Principle

When rainwater falls on the sensor, it creates a conductive path that triggers the 555 Timer IC. The timer activates a BC547 transistor, which drives the buzzer and LED, providing an immediate alert.
